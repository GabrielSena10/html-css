Principais formatações

Negrito/Destaque

Termo em negrito usando a tag </b>(não semântica)

Termo em destaque usando a tag </strong>(semântica)

Itálico/Ênfase

Termo em itálico usando a tag </i>(não semântica)

Termo em ênfase usando a tag </em>(semântica)

Texto grande e pequeno

Temos um texto grande utilizando a tag </big>

Temos um texto pequeno utilizando a tag </small>

Texto deletado e inserido

Podemos marcar um texto como deletado para indicar que ele deve ser lido mas não considerado com a tag </del>

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois com a tag </ins>(semântico)

Temos o sublinhado com a tag </u>(não semântico)

Texto sobrescrito

Temos por exemplo x20+3, usando a tag </sup>

Texto subscrito

Temos por exemplo H2O, usando a tag </sub>
